smooth_perm computes the number of smooth Schubert varieties in the decomposition of GLn/B, using a result of Lakshmibai-Sandhya, that gives an avoidance criteria pour for the smoothness of the Schubert variety X(w).
![resultat](nb_smooth_perms.png)

And length_bruhat_intervals computes the lengths of the Bruhat intervals of some classes of permutations.
